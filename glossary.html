Glossaire des commandes Git et Github

Cette section est en cours de construction. Nous ajoutons continuellement de nouveaux termes et commandes.

Commandes Basiques
git init
Initialise un nouveau dépôt git.
git clone
Clone un dépôt existant.
git add
Ajoute des fichiers à l'index.
Commandes Avancées
git rebase
Réapplique des commits sur une autre base.
git merge
Fusionne deux branches.
