    • Débogage avec Visual Studio Code

    Apprenez à utiliser les outils de débogage intégrés de Visual Studio Code pour améliorer votre flux de travail.

    Configuration du Débogage

    Suivez ces étapes pour configurer le débogage :

    1. Ouvrez le fichier à déboguer.
    2. Placez un point d'arrêt (breakpoint) en cliquant à gauche du numéro de ligne.
    3. Cliquez sur l'icône de débogage dans la barre latérale gauche et configurez votre environnement de débogage.
    Utilisation des Outils de Débogage

    Utilisez ces outils pour déboguer votre code :

    • Play (F5) - Démarre le débogage.

    • Pause - Met en pause l'exécution du programme.
    • Step Over (F10) - Exécute la ligne actuelle et passe à la suivante.
    • Step Into (F11) - Entre dans la fonction appelée à la ligne actuelle.
