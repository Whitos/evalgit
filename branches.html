Gestion des Branches Git

Explorez les commandes essentielles pour la gestion des branches dans vos projets Git.

Commandes de Base

Liste des branches locales et commandes pour la gestion des branches :

git branch - Liste toutes les branches locales.
git branch <nom de la branche> - Crée une nouvelle branche.
git checkout <nom de la branche> - Bascule sur la branche spécifiée.
git merge <nom de la branche> - Fusionne la branche spécifiée avec la branche courante.
Commandes Avancées

Commandes pour les opérations avancées de branche :

git branch -d <nom de la branche> - Supprime la branche localement.
git push origin --delete <nom de la branche> - Supprime la branche sur le dépôt distant.
