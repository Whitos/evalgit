    Automatisez votre workflow avec GitHub Actions.

    Création d'un Workflow

    Voici les étapes pour créer un workflow de base :

    name: CI
    on: [push, pull_request]
    jobs:
      build:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v2
          - name: Run a one-line script
            run: echo Hello, world!
    Gestion des Secrets

    Utilisez des secrets pour sécuriser vos informations sensibles :

    secrets:
      GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
      MY_SECRET: ${{ secrets.MY_SECRET }}
