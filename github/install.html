    • Installer GitHub

    Suivez ces étapes pour installer et configurer GitHub sur votre système.

    Téléchargement de GitHub Desktop

    Visitez desktop.github.com et téléchargez la version appropriée pour votre système d'exploitation.

    Configuration de Base

    Après l'installation, ouvrez GitHub Desktop et connectez-vous avec votre compte GitHub.

    Vérification de la Configuration

    Pour vérifier que tout est configuré correctement, assurez-vous que vos dépôts apparaissent dans l'interface.
