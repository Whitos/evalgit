    • Installer Git

    Suivez ces étapes pour installer et configurer Git sur votre système.

    Téléchargement de Git

    Visitez git-scm.com/downloads et téléchargez la version appropriée pour votre système d'exploitation.

    Configuration de Base

    Après l'installation, ouvrez votre terminal et configurez votre identité Git avec les commandes suivantes :

    git config --global user.name "Votre Nom"
    git config --global user.email "votre-email@example.com"
    Vérification de la Configuration

    Pour vérifier que tout est configuré correctement, tapez :

    git config --list
