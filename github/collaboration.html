    • Collaboration et Pull Requests

    Découvrez comment collaborer efficacement sur GitHub avec les Pull Requests.

    Création d'une Pull Request

    Suivez ces étapes pour créer une Pull Request :

    git checkout -b feature-branch - Créez et basculez vers une nouvelle branche.
    git push origin feature-branch - Poussez vos modifications vers GitHub.
    Ouvrez une Pull Request sur GitHub - Utilisez l'interface GitHub pour ouvrir une nouvelle Pull Request.
    Revue et Merge des Pull Requests

    Suivez ces étapes pour revoir et merger une Pull Request :

    • Accédez à l'onglet "Pull Requests" de votre repository sur GitHub.

    • Sélectionnez la Pull Request que vous souhaitez revoir.
    • Utilisez l'interface pour laisser des commentaires, approuver ou demander des modifications.
    • Une fois approuvée, utilisez le bouton "Merge" pour fusionner la Pull Request.
