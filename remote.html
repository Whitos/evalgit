Commandes Git Distantes

Gérez vos dépôts distants avec ces commandes essentielles.

Ajouter et Gérer les Dépôts Distantes

Voici les commandes pour ajouter, modifier, et supprimer des dépôts distants :

git remote add origin <URL du projet>
git remote set-url origin <nouvelle URL>
git remote remove origin
Push et Pull

Envoyez et recevez des changements entre vos dépôts locaux et distants :

git push origin <nom de la branche>
git pull origin <nom de la branche>
